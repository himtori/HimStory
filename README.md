## 게임 소개

*기억을 잃은 한 배우가 있다.*

'힘스토리'는 대학로를 배경으로 한 선택지 게임이며, 짧은 러닝타임 안에 플레이어의 선택에 따라 여러 엔딩을 즐길 수 있도록 디테일하게 구성되었습니다.

### 게임 스토리
기억을 잃은 한 배우를 마주친 플레이어,

과연 직접 기억을 찾아줄 수 있을까?
### 게임 포인트
엔딩별로 추가되는 고퀄리티 일러스트를 수집해보세요.

## 지원 플랫폼
### PC (MAC, Windows)
#### 실행
1. [다운로드](https://drive.google.com/drive/folders/1g90i1uhFjZJpGWORu4M-tGC8On1hWPR2?usp=drive_link) 링크로 들어갑니다.
2. 실행시킬 pc의 os에 맞는 압축파일을 다운로드합니다.
   <p align="center">
   <img src =   "https://github.com/himtori/HimStory/blob/main/%E1%84%83%E1%85%B3%E1%84%85%E1%85%A1%E1%84%8B%E1%85%B5%E1%84%87%E1%85%B3%20%E1%84%89%E1%85%A1%E1%84%8C%E1%85%B5%E1%86%AB.png" width="90%">
   </p>
3. 압축을 풉니다.
4. 실행파일을 실행시킵니다.

### 안드로이드
1. [다운로드](https://drive.google.com/drive/folders/1X1HKINzSHUvjkiZVG1TI8J2j2F4W5DMX?usp=sharing) 링크로 들어갑니다.
2. apk 파일을 다운로드합니다.
3. 실행시킵니다.
4. 설치를 누릅니다.
5. 설치된 앱을 실행시킵니다.

### ios
1. 앱스토어에서 힘스토리 검색합니다.
2. 설치 후 실행합니다.
